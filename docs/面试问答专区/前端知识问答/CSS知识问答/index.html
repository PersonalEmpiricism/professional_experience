CSS有哪些选择器,选择器的优先级​

  • 基本选择器
    • ID选择器
    • 标签选择器(也叫元素选择器)
    • 类选择器
    • 伪类选择器(:hover,:active)
    • 属性选择器
    • 通配符选择器
  • 复合选择器
    • 父子选择器(跟结构相关)
    • 后代选择器(跟结构相关)
    • 兄弟选择器(跟结构相关)
    • 交集选择器(条件选择器)
    • 并集选择器(也叫组合选择器或群组选择器)

CSS选择器优先级最高到最低顺序​

  1. id选择器（#myid）
  2. 类选择器（.myclassname）
  3. 标签选择器（div,h1,p）
  4. 子选择器（ul < li）
  5. 后代选择器（li a）
  6. 伪类选择（a:hover,li:nth-child）
